# 발주공고 분석 대시보드

경영진용 발주공고 통합 분석 대시보드 + AI 채팅형 조건 설정 마법사

## 파일 구조

```
발주공고분석/
├── index.html      # 메인 대시보드 (KPI, 히트맵, 데이터테이블)
├── wizard.html     # AI 채팅형 조건 설정 마법사
├── data.js         # 데이터 레이어 (현재 랄랄라.xlsx 기반, DB 연동 예정)
├── 랄랄라.xlsx     # 원본 데이터
└── README.md
```

## 주요 기능

### 대시보드 (`index.html`)
- **KPI 카드**: 총 공고금액, 총 공고수, 최다 발주기관, 특수실적 건수
- **종목별 공고 분포 차트** (실제 데이터 기반 자동 집계)
- **계약방법 분포 차트** (적격심사 vs 수의)
- **지역/종목 교차 히트맵** (실제 공고 데이터 기반)
- **공고 상세 목록**: 정렬/필터 가능
- **상세 필터 모달**: 지역, 종목, 계약방법, 특수실적 여부

### AI 마법사 (`wizard.html`)
- **채팅형 UI**: AI 메시지 → 사용자 선택 → 다음 단계
- **순차 조건 설정**: 종목 → 지역 → 발주처 → 금액범위 → 계약방법
- **URL 파라미터**로 대시보드에 조건 자동 전달

## 데이터 구조 (랄랄라.xlsx)

| 컬럼 | 설명 |
|------|------|
| 공고번호 | 공고 식별자 |
| 추정가격 | 추정 계약 금액 (원) |
| 지역제한 | 입찰 가능 지역 |
| 종목 | 공사 종목 |
| 특수실적 | 필요 특수실적 (없으면 null) |
| 계약방법 | 적격심사 / 수의 |
| 적격평가세부기준 | 적격심사 세부 기준 (발주처 식별에 활용) |
| 입력일 | 데이터 입력 날짜 |

## DB 연동 계획

현재 `data.js`의 `RAW_DATA` 배열을 API 호출로 교체 예정:

```javascript
// 현재 (정적)
const RAW_DATA = [...];

// 추후 (API 연동)
const RAW_DATA = await fetch('/api/announcements').then(r=>r.json());
```

## 로컬 실행

파일 탐색기에서 `index.html`을 브라우저로 열거나:

```bash
npx serve .
```
